    Производственная линия

    Процессы

    • Pre Assembly

      Резка пластин. GBD (Grinding Before Dicing, утонение перед резкой). DBG (Dicing Before Grinding — резка перед утонением).

    • Attach Print

      Монтаж кристалла на подложку. После установки кристаллов на подложку происходит процесс Attach Cure — обработка в сушильной печи, в процессе которой происходит полимеризация адгезива.

    • Wire Bond

      Процесс микросварки. Для создания соединений используется оборудование ультразвуковой термокомпрессионной микросварки.

    • Mold

      Автоматическая формовочная система. Она предназначена для всех существующих и будущих типов упаковки с использованием любого типа гранулированного формовочного компаунда.
